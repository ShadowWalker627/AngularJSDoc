Angular内置function

angular.bind 相当于 apply 和 call

angular.bind(self, fn, args)

  • ①、self：fn要运行的上下文作用域
  • ②、fn：要操作的函数
  • ③、args：要传入的参数
angular.bootstrap

angular.bootstrap(element, [modules], [config])

用于手动启动 angular 应用

  • ①、element：作为 angular 应用根节点的DOM 元素
  • ②、[modules]：要加载到应用中的模块数组
  • ③、config：object 用于定义应用程序配置选项的对象，其属性strictDi,默认值为 false，是否禁用应用程序的自动函数注释
angular.copy

angular.copy(source, [destination])

深度拷贝数组或对象，source 可以是 undefined 或者 null

source：源，destination：目标

destination的类型必须与source类型相同

  • 如果未提供destination，那么会创建source(对象/数组)的副本
  • 如果提供source，则删除destination的所有元素(array)或者所有属性(object)， 然后将source中的目标元素/属性复制到destination中
  • 如果source不是数组或者对象(例如 null和 undefined)，则返回source
  • 如果source与destination相同，则抛出异常
angular.element angular.element(element)
  • 如果jQuery 可用，angular.element是jQuery函数的别名
  • 如果jQuery不可用，angular.element作为Angular的内置jQuery子集，称为jQuery lite 或者jqLite

注意：Angular中的所有元素引用都是jQuery或者 jqLite包装(如指令的 compile 或者 link中的 element参数)
angular.equals

angular.equals(o1, o2)

确定两个对象或两个值是否相等，支持值类型，正则表达式，数组和对象；返回值为 Boolean

以下情况，返回 true：

  • 两个参数(对象/值)通过"==="比较返回 true
  • o1/o2的类型相同，属性值都相同
  • 两个NaN 返回 true，JavaScript中NaN == NaN 返回 false
  • 两个值表示相同的正则表达式返回 true，JavaScript 中/abc/ == /abc/ 返回 false
angular.extend

angular.extend(dst, src)

将 src 的可枚举属性复制到 dst

可以指定多个src

如果要保留原始对象，将空对象作为 dst

注意：angular.extend不支持递归合并(深度复制)
angular.forEach

angular.forEach(obj/array, iterator, [context])

不迭代继承的属性，因为使用hasOwnProperty方法过滤

如果 obj/array为 undefined或 null，不会抛出错误
