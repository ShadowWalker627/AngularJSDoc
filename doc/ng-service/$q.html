异步编程--$q 实现

    调用$q.defer()生成一个deferred实例：
        具有三个方法：notify、resolve、reject
        一个属性：promise，
            属性值是Promise的实例：有三个方法：
                then(successCallBack,[errorCallback], [notifyCallback])
    $http 返回值为一个 promise，可形成promise链
        promiseB = promiseA.then(function(result) {
            return result + 1;
        });
