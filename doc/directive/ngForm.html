ngForm

    ngSubmit与ngClick 只能有一个，来避免重复提交表单，如果同时设置，点击 enter 回车时，会同时出发这两个
    ngClick 在模型更新之前触发
    ngSubmit 可访问已更新的模型

    input：ng 属性：
        ng-model
        name
        required
        ng-required
        ng-minlength
        ng-maxlength
        ng-pattern
        ng-change
        ng-trim
    表单属性：
        $pristine
        $dirty
        $valid
        $invalid
        $pending
        $submitted
        $error：除 error外，其他属性值都为 boolean。error 值为 object

    需要在 controller 中获得FormController 的以上表单属性，需要将 form 的 name 属性绑定到controller别名
userType: 
    .my-form{
        transition: all linear 0.5s;
        background: transparent;
    }
    .my-form.ng-invalid{
        background: red;
    }
userType = {{vm.userType}}
myForm.$valid = {{myForm.$valid}} myForm.$error = {{myForm.$error}}
